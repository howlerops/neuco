## Competitive Positioning (updated 2026-03-13)

**Core moat:** No competitor does signal→spec→code→PR end-to-end for teams. Feedback tools stop at insights. Code gen tools require manual prompts. Factory.ai does full SDLC but not feedback-driven.

**Closest competitor:** Factory.ai ($50M Series B, 200% QoQ growth) — agent-native SDLC orchestration. They do GitHub/Linear/Slack integration + autonomous agents. But they don't ingest customer feedback or do synthesis. More of a "developer productivity" play than "product intelligence."

**Signal/Insight tools:**
- Enterpret — launched agentic platform (Oct 2025). Wisdom AI Agent for Q&A, Action Agents for bug detection/churn risk. Moving toward actions, not just insights. Enterprise-only.
- Dovetail — AI-first platform with agents + automation. Alloy/Salesforce/Linear/Gong integrations. 236% ROI independent study. Enterprise-focused.
- Productboard ($19-59/maker/mo + $20 AI add-on) — launched Spark (context-aware spec gen). Closest to adding code gen but still 12-18 mo away.
- Unwrap.ai — $12M Series A (Jan 2025), 500% YoY, 3000+ integrations. Not agentic yet.
- Canny — AI Autopilot for summarization. $79-359/mo. Simple, not enterprise-grade.
- UserVoice — declining, $999/mo, outdated UX.

**Code gen tools:**
- Lovable — $400M ARR (March 2026), $6.6B valuation. Full-stack from prompts. BUT security incident Feb 2026 exposed 18K users. Enterprise traction (Klarna, Uber).
- Bolt.new — $40M ARR, Bolt Cloud for hosting/auth/DB. 1M+ websites. Good prototyping DX.
- v0 by Vercel — expanded beyond UI to agentic. Sandbox runtime, GitHub imports. Broader audience now.
- Cursor — $9.9B valuation, 50% Fortune 500. IDE-bound. Bugbot for code review. Agent-centric.
- GitHub Copilot Workspace — Mission Control for multi-agent. But slow startup (90+ sec), UX concerns.

**Market dynamics:**
- TAM: ~$800M-$1.2B (product intelligence + AI code gen intersection)
- Claude Code: 75% usage in 5-20 person startups, 46% "most loved"
- 86% of leaders uncertain which AI tools actually pay off — ROI clarity is a selling point
- Security is now a differentiator post-Lovable incident
- No dominant signal-to-PR product exists — Neuco's opening

**Threats (updated):**
- Factory.ai could add feedback ingestion (they have agent infra) — 6-12 mo risk
- Productboard Spark + Lovable combo is strong for fast teams
- Enterpret/Dovetail agents might start generating code directly — 12-18 mo risk
- GitHub Copilot Workspace + Dovetail integration — 18 mo risk

**Strategic implications:**
- Security/reliability > speed as a selling point (post-Lovable incident)
- Speed to first PR must beat stringing Productboard + Lovable + GitHub together
- Enterprise integrations (Linear, Slack, Sentry, Notion) are table stakes — copy Factory's playbook
- 5-50 person engineering-led teams using Claude Code are the sweet spot

**Why:** Informs roadmap priorities and positioning.
**How to apply:** When evaluating features, prioritize: (1) security/reliability of generated code, (2) speed of feedback-to-PR loop, (3) enterprise integration depth, (4) team-first workflows. Watch Factory.ai closely.
